Features




Learn

Students get to learn from the videos created by educators. All the videos are separated by topics

Review

Students get to review what they have learned with flashcards

Evaluate

Finally there are assessment quizzes created by educators to test students knowledge


Create

Educators can link their videos, create flashcards and quizzes for their students

Structure

Educators can organize their content into courses with the help of course codes, which students will use for enrolling

Evaluate

Educators can review student progress via scoreboards

FAQs

The code is for identifying the course(s) a student is enrolled in, it is provided by educators.

Select/enroll course from the student home page and switch to the other pages (video, flashcards, quiz) to view course content.

The course code is for structuring your content into courses and letting the students have access to each course separately.

Select/Add course from the educator home page and switch to the other pages (video, flashcards, quiz) to manage course content.
